# Laravel 8 Rest Api

Studi kasus aplikasi surat menyurat

### Langkah Instalasi

- Update composernya dulu
```sh
composer update
```

- Copy env
```sh
cp .env.example .env
```

- Buat database dan atur file `.env` nya

- Migrasi database
```sh
php artisan migrate
```

- Seeding data awal
```sh
php artisan db:seed
```

- Buat simbolik link
```sh
php artisan storage:link
```
